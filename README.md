# Streamlit - Perbandingan Model

Cara menjalankan aplikasi Streamlit untuk membandingkan `hasil_simplernn2.json`, `hasil_gru2.json`, dan `hasil_lstm2.json`:

1. (Opsional) Buat virtual environment dan aktifkan.
2. Install dependensi:

```bash
pip install -r requirements.txt
```

3. Jalankan aplikasi:

```bash
streamlit run uas_app.py
```

File JSON harus berada di folder yang sama dengan `streamlit_app.py`.
